Descripción del Proyecto

Este proyecto es un pipeline sencillo de ingeniería de datos que realiza las siguientes tareas:

  • Lee un archivo CSV con datos ficticios de ventas.
  • Realiza limpieza y transformaciones básicas con la librería pandas de Python.
  • Carga los datos limpios en una base de datos PostgreSQL local para permitir consultas y generación de reportes simples.
  • Ideal para aprender el flujo básico ETL (Extracción → Transformación → Carga) en una sola máquina.

Ejemplo de Código (pandas para transformación)

import pandas as pd

# Leer CSV
df = pd.read_csv('data/raw/sales.csv')

# Limpieza básica
df = df.dropna()  # eliminar filas con datos faltantes

# Transformación: crear columna de total
df['total'] = df['quantity'] * df['price']

# Mostrar primeros registros
print(df.head())

Base de Datos PostgreSQL

Los datos transformados se cargan en una base de datos PostgreSQL local. Aquí puedes realizar consultas para obtener reportes simples, por ejemplo: 

SELECT product_name, SUM(total) AS total_ventas
FROM ventas
GROUP BY product_name
ORDER BY total_ventas DESC;
